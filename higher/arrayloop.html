What is for loop for arrays

A for loop for an array is used to access and perform actions on every element inside the array.
A for loop for arrays is used to go through each element in the array step by step and perform some operation with it.

What is Nested loop

nested loop is a looping structure where one loop runs inside another loop.A
The inner loop runs completely each time the outer loop runs once.

Why we use nested loop

To handle multi-dimensional data (like rows and columns).
To compare elements from different arrays.
To repeat inner actions multiple times within an outer condition
